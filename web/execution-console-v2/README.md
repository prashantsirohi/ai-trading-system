# Execution Console V2

This directory hosts a parallel operator UI and does not replace the existing `web/execution-console` app.

## App
- `/Users/prashant/my-ai-project/trading_system/ai-trading-system/web/execution-console-v2/ai-trading-dashboard-starter`

## Quick Start
```bash
cd /Users/prashant/my-ai-project/trading_system/ai-trading-system/web/execution-console-v2/ai-trading-dashboard-starter
npm install
npm run dev
```

## Notes
- Mock mode is enabled by default (`VITE_USE_MOCK_API=true`).
- Set `VITE_USE_MOCK_API=false` in `.env` to call `/api/execution/*` endpoints.
- The existing console at `/Users/prashant/my-ai-project/trading_system/ai-trading-system/web/execution-console` remains unchanged.
